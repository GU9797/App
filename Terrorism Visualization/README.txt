#Terrorism in the U.S.

For our final interactive data visualization project, we mapped Terrorism occurrences in the U.S. from 1970 to 1997.
Among the files submitted, those written to create the final product include:

/scaffold/index.html
Code for html elements.
Main project implementation.

/scaffold/src/components/map-component.js
Code for mapbox implementation.

Visualization Preview.jpg
Screenshot of visualization.

writeup.pdf
Report on conclusions drawn from visualization.

To run visualization, run 'yarn' in scaffold directory, then run 'yarn start'.
